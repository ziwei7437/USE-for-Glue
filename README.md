# USE-for-Glue

Universal Sentence Encoder <https://tfhub.dev/google/universal-sentence-encoder/4> for Glue Tasks.


No Fine-tuning tasks implemented on the Encoder. We only test the following tasks:


* MRPC

* QQP

* MNLI

* SNLI

* QNLI

* RTE


Requirements:
    Pytorch Version: 1.4
    Tensorflow-gpu: 2.1
